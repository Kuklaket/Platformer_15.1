# Platformer_15.1
Теперь враги и игрок могут атаковать друг друга, когда подходят близко. 
Враги начинают преследовать игрока, когда видят его, в обратном случае они патрулируют территорию. Также игрок может лечиться, подбирая аптечки, которые случайным образом разбросаны по уровню.

Отобрение здоровье в этой задаче нет.
